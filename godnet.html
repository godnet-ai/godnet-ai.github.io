What is a Neural Network(NN)?

Remember the simple function we talked about: f (x) = 2x² + 3x + 1 = y. Here the 
input x goes through a transformation and produces y. Let’s call this one layer of 
computation. What if we apply some other transformation on this output y and 
convert it into some z. Thus the overall transformation that x went through to 
produce z is much more complex than the transformation required to produce y.

NN pretty much follows the same concept. It’s are just a bunch of nodes connected 
layer by layer which defines the transformation that the input would go through to 
produce the output. A specific connection (or network) of layers and nodes is 
called an Architecture.
Neural Networks: Nodes and layers define the rules to turn inputs to outputs

Each node has some ‘parameters’ associated with it, which are used to perform  
the transformations. These parameters are learned using some past data 
(called training data) and a learning algorithm (like Gradient Descent). Just like 
you would adjust the pegs to tune the string of a guitar until you hit the perfect 
note, we adjust the ‘parameters’ of a neural network until it has learned how to 
perform the task that you want it to perform.


Convolutional Neural Networks— CNNs


The above shown neural network is known as a ‘Multi-Layer Perceptron’. 
However, there are other variants of neural networks that are suited for specific
tasks. Convolutional Neural Networks (CNNs) are one such class of neural networks 
that are most commonly applied to analyze visual imagery. CNN's work by extracting 
a more and more abstract representation of the features in an image by passing 
the image through convolutional layers. This representation is learned by adjusting 
the parameters associated with the layers such that it satisfies some given training
